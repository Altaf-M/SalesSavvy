# SalesSavvy

Full Stack E-Commerce Platform built using Spring Boot, JWT, React, and MySQL.

## Tech Stack

- Frontend: React (Vite)
- Backend: Spring Boot
- Database: MySQL
- Authentication: JWT

## Features

- User Registration
- Role-Based Access Control (RBAC)
- Secure Authentication with JWT
- RESTful API Integration
- Responsive UI
